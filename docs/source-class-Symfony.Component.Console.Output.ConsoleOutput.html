 18:  *

 24:  *

 28:  *
 29:  * @api
 30:  */

 39:      * @param bool|null                     $decorated Whether to decorate messages (null for auto-guessing)
 40:      * @param OutputFormatterInterface|null $formatter Output formatter instance (null to use default OutputFormatter)
 41:      *
 42:      * @api
 43:      */
